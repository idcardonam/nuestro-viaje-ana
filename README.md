# Nuestro Viaje · Ana & Iván 🤍

Sitio web sorpresa para el viaje al Caribe (27 Jun — 3 Jul 2026).

**URL en vivo:** https://idcardonam.github.io/nuestro-viaje-ana/

## Archivos

| Archivo | Uso |
|---------|-----|
| `index.html` | Sitio principal (QR apunta aquí) |
| `parasol-qr.html` | Imprimir media carta → pegar en parasol |
| `nota-guantera.html` | Imprimir media carta → guantera del carro |
| `css/style.css` | Estilos del sitio |
| `js/app.js` | Checks, fotos, alertas GPS, sorpresa |
| `img/` | Fotos de la pareja |

## Imprimir

1. Abre `parasol-qr.html` en Chrome → Imprimir → **Media carta (A5)** → Sin márgenes
2. Abre `nota-guantera.html` → Imprimir → **Media carta** → Sin márgenes

## Sorpresa secreta

- Toca el corazón 🤍 del hero **3 veces**, o
- Completa ~35% de los checks de la Ruta

## Datos en el celular

Las fotos y checks se guardan en `localStorage` del navegador. Para resetear: abre consola y ejecuta `ViajeApp.reset()`.
